## 技術堆疊

### 前端
- **React 18+**: UI 框架
- **TypeScript**: 型別安全
- **Tailwind CSS**: 樣式框架
- **Vite**: 建置工具

### 後端
- **ASP.NET Core 8+**: Web API 框架
- **C# 12**: 程式語言
- **Entity Framework Core**: ORM
- **MS SQL Server 2022+**: 資料庫
- **Redis**: 快取

### 容器化
- **Docker**: 容器化平台
- **Docker Compose**: 多容器編排

## 後續開發步驟

### 1. 環境設定

#### 前端
```bash
cd frontend
npm install
npm run dev
```

#### 後端
```bash
cd backend
dotnet restore
dotnet run --project WarehouseManagementSystem.Api
```

#### Docker
```bash
docker-compose -f docker/docker-compose.yml up -d
```

### 2. 開始第一個模組開發

根據 SDD 工作流程，開始第一個模組的開發：

1. **Phase 1: SPEC** - 在 `1.specs/` 建立規格文件
2. **Phase 2: PLAN** - 在 `2.plans/` 建立計畫文件
3. **Phase 3: TASKS** - 在 `3.tasks/` 建立任務文件
4. **Phase 4: EXECUTE** - 開始實作程式碼
5. **Phase 5: REVIEW** - 在 `4.reviews/` 建立驗收文件

### 3. 建議優先開發的模組

根據設計規範，建議優先開發：

1. **系統管理作業** - 建立基礎的權限與使用者管理
2. **基本作業** - 建立基礎資料維護功能
3. **登入與首頁** - 建立系統入口與 Dashboard
